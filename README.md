# Sona College of Technology - Learning Hub

This project is a professional UI recreation of the Sona College of Technology website, built with React, Tailwind CSS, and Vite.

## Development

1. **Install Dependencies**:
   ```sh
   npm install
   ```

2. **Run Development Server**:
   ```sh
   npm run dev
   ```

3. **Build for Production**:
   ```sh
   npm run build
   ```

## Technologies

- **React**
- **Vite**
- **Tailwind CSS**
- **shadcn-ui**
- **TypeScript**
